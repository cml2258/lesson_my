- 动态规划
    - 斐波那契数 （通常用 F(n) 表示）形成的序列称为 斐波那契数列 。
        该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。
        解法:
        1. 暴力递归 时间复杂度高 自顶向下
        2. 动态规划：状态转移方程f(n)=f(n-1)+f(n-2)
            - 递归+记忆 
            - 自底向上 dp数组
                开辟一个数组空间，循环缓存每一个计算的值dp[i]
            - 优化：动态数组 
                只需要dp[0],和dp[1],加上一个temp变量，缓存加和后 交换结果
            - 继续优化： 三个变量 -> 过去状态 当前状态 未来状态 
                过去状态 f(n-2)
                当前状态 f(n-1)
                未来状态 f(n)=f(n-1)+f(n-2)
