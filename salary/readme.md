# js数据类型
    简单数据类型
        - number 不分长整，短整，浮点，都是数值类型
        - string
        - boolean
        - null  空 定义了，但值为空
            为空，（确定的类型），值为空
        - undefined 未定义 声明了变量（js允许只声明变量），但是没有赋值
            内存空间（undefined有地址，地址的长度不定，不定类型） js是弱类型语言

    复杂数据类型
        - object
            除了简单数据类型，都是对象
            array[] 可遍历对象
            object{} 对象字面量（可描述的）
            function(){} 函数是可执行的对象

        - 自学作业 typeof
            可以获得变量的类型，但是不准确
